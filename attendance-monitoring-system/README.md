# Attendance Monitoring System

Mobile + backend attendance management system designed for educational institutions.

## Tech Stack
React Native (Expo)
Node.js
Express.js
MongoDB
JWT Authentication

## Features
- Role-based access (Admin, Faculty, Student)
- Attendance marking and editing
- Attendance analytics
- Leave management
- Notifications
- CSV/PDF export reports

## Architecture
Mobile App (React Native) → REST API (Node.js + Express) → MongoDB Database

## Status
Portfolio case study project
