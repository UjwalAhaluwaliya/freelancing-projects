# Strategic Portfolio Rationalization System (SPRS)

AI-powered portfolio decision platform that helps organizations analyze projects and generate strategic recommendations.

## Tech Stack
React.js
Python Flask
MongoDB
scikit-learn
pandas
numpy
PuLP

## Features
- Project portfolio management
- AI-driven project recommendations
- Duplicate project detection
- Risk anomaly detection
- Budget optimization using linear programming
- Analytics dashboard with charts

## ML Pipeline
AHP → Business scoring  
Random Forest → Success prediction  
K-Means → Project clustering  
TF-IDF + Cosine Similarity → Duplicate detection  
Isolation Forest → Risk anomaly detection  
Linear Programming → Budget optimization

## Architecture
React Frontend → Flask REST API → MongoDB Database → ML Analysis Engine
