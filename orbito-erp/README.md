# Orbito ERP

AI-powered HR and ATS platform designed to automate hiring workflows and HR operations.

## Tech Stack
React
FastAPI
Supabase (PostgreSQL)
JWT Authentication
n8n automation
Gemini AI

## Features
- HR management system
- Applicant Tracking System (ATS)
- Candidate and job management
- Hiring workflow automation
- Leave management
- Notification system

## AI Tools
- Policy chatbot
- Resume scoring
- Job description generator

## Architecture
React Frontend → FastAPI Backend → Supabase Database → AI Services

## Status
Production-style ERP platform case study
