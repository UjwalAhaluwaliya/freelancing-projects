# Orbito ERP Frontend

## Setup

1. Copy `.env.example` to `.env`.
2. Set `VITE_API_BASE_URL` to backend URL.
3. Install deps:
   - `npm install`
4. Start dev server:
   - `npm run dev`

## Available Routes

- `/login`
- `/dashboard` (role-aware)
- `/hr/ats`
- `/hr/interviews`
- `/hr/leaves`
- `/hr/ai`
- `/admin/users`
- `/employee/leave`
- `/notifications`
