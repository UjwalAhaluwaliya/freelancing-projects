# Smart Logistic

Full-stack logistics and shipment management platform designed to streamline dispatch operations and delivery tracking.

## Tech Stack
Next.js
React
TypeScript
FastAPI
Supabase
PostgreSQL

## Features
- Shipment lifecycle management
- Dispatch assignment workflows
- Delivery status tracking
- Role-based dashboards
- Secure authentication
- Operational analytics

## Architecture
Next.js Frontend → FastAPI Backend → Supabase PostgreSQL Database
