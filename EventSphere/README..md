# EventoraX

Role-based event management and judging platform for competitions and talent events.

## Tech Stack
Next.js
React
TypeScript
Tailwind CSS
FastAPI
Supabase

## Features
- Event creation and management
- Judge assignment
- Participant registration
- Criteria-based scoring
- Automated result publishing
- Role-based dashboards

## Architecture
Next.js Frontend → FastAPI Backend → Supabase Database

## Status
Portfolio event evaluation system
