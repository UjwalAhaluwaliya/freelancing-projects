# Fittrine – Find Your Fitness Partner

A cross-platform fitness matchmaking app that connects users with compatible workout partners.

## Tech Stack
Flutter
Firebase Auth
Cloud Firestore
Firebase Messaging
Cloudinary

## Features
- User authentication
- Profile creation
- Workout preference matching
- Swipe-based partner discovery
- Real-time chat
- Workout scheduling
- Progress tracking
- Location-based filtering

## Architecture
Flutter Mobile App → Firebase Backend → Firestore Database

## Status
Portfolio mobile fitness platform
