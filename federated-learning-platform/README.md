# Federated Learning Platform

A privacy-preserving machine learning system where multiple clients collaboratively train models without sharing raw data.

## Tech Stack
Python
PyTorch
TensorFlow
Flower
FastAPI
Docker

## Features
- Federated client-server training architecture
- Local model training on client devices
- Secure aggregation of model updates
- Multiple communication rounds
- Training metrics tracking
- Experiment reproducibility

## Architecture
Client Nodes → Federated Server → Global Model Aggregation

## Impact
Supports privacy-preserving AI for domains such as healthcare, finance, and IoT.
