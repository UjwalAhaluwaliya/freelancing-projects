# AI Parent Monitoring System

A parental control platform with separate Parent and Child mobile apps for monitoring screen time and safe browsing.

## Tech Stack
Flutter
FastAPI
MongoDB
JWT Authentication
bcrypt password hashing

## Features
- Parent and child role authentication
- Screen time limit control
- Safe browsing with adult keyword filtering
- Child activity monitoring
- Usage reports and analytics
- Alerts for unsafe websites

## Architecture
Flutter Mobile Apps → FastAPI Backend → MongoDB Database

## Impact
Provides parents with real-time monitoring and safety tools for children's digital activity.

## Status
Portfolio mobile security platform
